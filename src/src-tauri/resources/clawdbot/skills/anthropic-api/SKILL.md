# Anthropic API

Call the Anthropic API for Claude-powered text generation and analysis.

## When to activate

- User asks to use Claude for a specialized sub-task
- User wants to analyze long documents or complex data
- User needs multi-turn reasoning for a specific problem

## Setup

Requires an `ANTHROPIC_API_KEY` environment variable.

## Capabilities

- Long-context document analysis (up to 200K tokens)
- Multi-step reasoning and planning
- Code generation and review
- Structured data extraction
