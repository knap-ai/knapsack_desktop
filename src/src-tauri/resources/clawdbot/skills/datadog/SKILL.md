# Datadog

Interact with Datadog for monitoring, alerting, and observability.

## When to activate

- User asks to check infrastructure or application metrics
- User wants to query logs or traces
- User needs to manage monitors or alerts
- User asks about dashboards or service health

## Setup

Requires a `DD_API_KEY` environment variable (API key from Datadog organization settings).

## Capabilities

- Query metrics, logs, and traces
- Manage monitors and alerts
- List and inspect hosts and containers
- Search and filter events
- View service maps and dependencies
